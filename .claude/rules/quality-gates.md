# Quality Gates

## Validate at EVERY checkpoint. Never skip quality checks.

## Content Length Minimums
| Deliverable | Min Words |
|-------------|-----------|
| VSL Scripts | 3000 |
| Sales Pages | 2000 |
| Case Studies | 1500 |
| Blog Posts | 1200 |
| YouTube Scripts | 1500 |
| Newsletters | 800 |
| Press Releases | 500 |
| Email (each) | 300 |
| LinkedIn Posts | 150-3000 chars |

## Output Quality Rules
| File Pattern | Min Words | Min Sections | Required Keywords |
|-------------|-----------|--------------|-------------------|
| `*vsl*.md` | 2000 | 8 | Hook, Problem, Solution, Offer, CTA |
| `*email*.md` | 300 | 3 | Subject, CTA |
| `*research*.md` | 1500 | 5 | Summary, Sources, Findings |
| `*report*.md` | 2000 | 5 | Summary, Recommendations |
| `*blog*.md` | 1000 | 4 | Introduction, Conclusion |
| `*sales*.md` | 1500 | 6 | Headline, Problem, Solution, CTA |
| `*.md` (default) | 500 | 3 | — |

## Validation Checklist
- [ ] Required fields present?
- [ ] Output format correct (markdown, JSON)?
- [ ] Word count meets minimum?
- [ ] No API errors in output?
- [ ] No placeholder text remaining?
- [ ] No PII/sensitive data exposed?
- [ ] Links are real (no placeholder URLs)?
- [ ] CTA present in marketing content?

## Research Quality
- Minimum 5 sources
- Data points with specifics (numbers, dates)
- Competitor analysis included
- No hallucinated facts
