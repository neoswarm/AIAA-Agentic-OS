## EMERGENCY TROUBLESHOOTING

### If CTR < 3%
- Headlines not compelling enough
- Ad copy doesn't match search intent
- Add more specific keywords
- Test emotional triggers

### If Conversion Rate < 5%
- Landing page issues (speed, relevance, trust)
- CTA not clear enough
- Form too long
- Phone number not prominent

### If CPA > $400
- Too broad targeting
- Landing page not optimized
- Wrong audience/keywords
- Need more negative keywords
