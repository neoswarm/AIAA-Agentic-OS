    1. **HTML** -- Add a client selector group between the skill header (`.skill-header` div, ends around line 510) and the mode tabs (`.mode-tabs` div, line 512). Insert this HTML:

       ```html
               <option value="">No client (general)</option>

       </div>
       ```

    2. **CSS** -- No new CSS needed. The `.field-group`, `.field-label`, `.field-select`, `.field-tooltip`, and `.field-hint` classes already exist in skill_execute.html's `<style>` block.

    3. **JavaScript** -- Add a `loadClientSelector()` function in the script IIFE:
       ```javascript
       async function loadClientSelector() {
           try {
               var data = await fetchAPI('/api/v2/clients');
               var clients = data.clients || [];
               if (clients.length === 0) return; // Hide selector if no clients

               var select = document.getElementById('client-selector');
               clients.forEach(function(client) {
                   var opt = document.createElement('option');
                   opt.value = client.slug;
                   opt.textContent = client.name;
                   select.appendChild(opt);
               });
               document.getElementById('client-selector-group').style.display = 'block';

               // Pre-fill from query params if present (for re-run with client)
               var params = getQueryParams();
               if (params.client) {
                   select.value = params.client;
               }
           } catch (e) {
               // Silently fail -- client selector is optional enhancement
               console.warn('Could not load clients:', e);
           }
       }
       ```
       Call `loadClientSelector()` at the start of the page init (alongside `loadSkillDetails()`).

    4. **Include client in form submission** -- In the `collectFormData()` function (line ~765), after collecting form fields, add the client selector value:
       ```javascript
       var clientSelect = document.getElementById('client-selector');
       if (clientSelect && clientSelect.value) {
           data.client = clientSelect.value;
       }
       ```

    Use `fetchAPI()` from main.js (NOT raw fetch). The client selector degrades gracefully -- if the API call fails or returns no clients, the selector stays hidden and the form works normally.
