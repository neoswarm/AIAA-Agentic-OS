  1. **List webhooks -- GET /api/webhook-workflows:**
     - Route: `@api_bp.route('/webhook-workflows', methods=['GET'])`
     - Decorator: `@require_auth('read')`
     - Call `models.get_workflows(workflow_type='webhook')` -- note this returns workflows with status='active' by default
     - ALSO call `models.get_workflows(workflow_type='webhook', status='paused')` to include disabled webhooks
     - Combine both lists
     - Build response array where each item has: `name`, `description`, `slug` (from `webhook_slug`), `enabled` (True if status == 'active'), `webhook_url` (f"{request.host_url.rstrip('/')}/webhook/{slug}"), `forward_url`, `slack_notify` (bool), `source` (from description or "Database")
     - Return `{"webhook_workflows": result}`
     - CRITICAL: webhooks.js line 16 reads `data.webhook_workflows` as an array. Line 20 checks `.enabled` boolean. Line 136 finds by `.slug`.

  2. **Register webhook -- POST /api/webhook-workflows/register:**
     - Route: `@api_bp.route('/webhook-workflows/register', methods=['POST'])`
     - Decorator: `@require_auth('write')`
     - Parse JSON: extract `slug` (strip, lowercase, replace spaces with hyphens), `name` (strip)
     - Validate: if not slug or not name, return 400
     - Check existing: `models.get_workflow_by_slug(slug)`
     - Call `models.upsert_workflow()` with:
       - `workflow_id=slug`
       - `name=name`
       - `description=data.get('description', f'Webhook: {slug}')`
       - `workflow_type='webhook'`
       - `status='active' if data.get('enabled', True) else 'paused'`
       - `webhook_slug=slug`
       - `forward_url=data.get('forward_url')`
       - `slack_notify=data.get('slack_notify', False)`
     - Build webhook_url: `f"{request.host_url.rstrip('/')}/webhook/{slug}"`
     - Return `{"status": "registered" if new else "updated", "slug": slug, "name": name, "webhook_url": webhook_url}` with status code 201 (new) or 200 (update)
     - CRITICAL: webhooks.js createWebhook (line 337) and updateWebhook (line 364) both call this endpoint

  3. **Unregister webhook -- POST /api/webhook-workflows/unregister:**
     - Route: `@api_bp.route('/webhook-workflows/unregister', methods=['POST'])`
     - Decorator: `@require_auth('write')`
     - Parse JSON: extract `slug`
     - If not slug, return 400
     - Look up workflow: `models.get_workflow_by_slug(slug)`
     - If not found, return 404
     - Delete: `models.delete_workflow(slug)` -- this does a soft delete (sets status='deleted')
     - Return `{"status": "unregistered", "slug": slug, "name": workflow['name']}`
     - CRITICAL: webhooks.js deleteWebhook (line 113) calls this endpoint

  Wrap each endpoint in try/except with 500 error response.

     - Decorator: `@require_auth('write')`
     - Parse JSON: extract `slug`
     - If not slug, return 400
     - Look up workflow: `models.get_workflow_by_slug(slug)`

       test_payload = {
           "test": True,
           "source": "dashboard-test",
           "timestamp": datetime.utcnow().isoformat(),
           "webhook_name": workflow['name']
       }
       try:
           resp = http_requests.post(test_url, json=test_payload, timeout=10)
